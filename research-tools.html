Automated Research Data Collection Tool

A custom-built AI-powered automation tool designed to streamline the collection and processing of research data from online sources. This tool leverages large language models and web scraping techniques to automate time-intensive data collection tasks for academic research projects.

How It Works

  • Source Identification: Automatically identifies and monitors relevant news sources, databases, and online platforms based on research parameters
  • Data Extraction: Uses web scraping and API integration to collect structured data from multiple sources simultaneously
  • AI Processing: Employs large language models to extract key information, classify events, and structure unstructured text data
  • Quality Control: Implements automated validation checks and deduplication to ensure data accuracy and consistency
  • Export & Integration: Outputs clean, structured datasets in various formats (CSV, JSON, Excel) for immediate analysis

Key Capabilities

Multi-Source Collection

Simultaneously monitors and extracts data from news websites, social media, and public databases

Multilingual Support

Processes content in Arabic, English, German, and Turkish with language-specific NLP pipelines

AI-Powered Analysis

Leverages LLMs for intelligent data extraction, classification, and event identification

PRO-TEST: Protest Violence Prediction

A machine learning proof-of-concept developed to predict the likelihood of violence at protest demonstrations. This educational project demonstrates the application of supervised learning to social science research questions.

  • Dataset: 13,000+ protest event records
  • Model: Random Forest classifier achieving 80% accuracy
  • Purpose: Le Wagon Data Science Bootcamp capstone project

Note: This is a proof-of-concept project developed for educational purposes, demonstrating technical proficiency in machine learning workflows.
